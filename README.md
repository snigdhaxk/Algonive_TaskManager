A clean, minimal task manager built to help you stay on top of your day — built during my Web Development Internship at Algonive.
 Live Demo
🔗 https://snigdhaxk.github.io/Algonive_TaskManager/

✨ Features
✅ Create Tasks — Add tasks with title, description, due date & priority
🟢🟡🔴 Priority Levels — Categorize tasks as Low, Medium, or High priority
📂 Smart Filtering — Switch between All, Active, Completed, and Overdue views
🔍 Search — Instantly search through your tasks
🔃 Sort Options — Sort by Due Date, Priority, Created Date, or Title
📊 Progress Tracker — Visual progress bar showing tasks done vs left
⚠️ Overdue Detection — Automatically flags tasks past their due date
🌑 Dark UI — Sleek dark theme for a focused, distraction-free experience
🛠️ Tech Stack
Technology	Usage
HTML5	Structure & semantic markup
CSS3	Styling, layout, dark theme
JavaScript (Vanilla)	Logic, DOM manipulation, state management
GitHub Pages	Deployment & hosting
